\section*{Страница 2: 7 главных сценариев работы с ИИ}

\subsection*{Сценарий 1: Работа с текстом}
\textbf{Что умеет:} писать, редактировать, переводить, резюмировать, переписывать.

\textbf{Примеры задач:}
\begin{itemize}
\item Написать email, статью, пост, стратегию, договор
\item Сократить 20-страничный отчёт до 1 страницы
\item Перевести технический текст на простой язык
\item Создать 10 вариантов заголовков для рекламы
\end{itemize}

\textbf{Промпт-пример:}
\begin{verbatim}
Перепиши этот текст для LinkedIn. Тон: экспертный,
но не занудный. Добавь конкретные цифры.
Длина: 150 слов. Цель: получить лиды.
[вставьте текст]
\end{verbatim}

\subsection*{Сценарий 2: Анализ данных и документов}
\textbf{Что умеет:} обрабатывать CSV, Excel, PDF, выдавать инсайты, строить прогнозы.

\textbf{Примеры задач:}
\begin{itemize}
\item Проанализировать P\u0026L за квартал и найти причины падения маржи
\item Сравнить 3 предложения поставщиков — плюсы, минусы, риски
\item Найти аномалии в платёжных операциях
\item Спронозировать выручку на следующий квартал
\end{itemize}

\textbf{Промпт-пример:}
\begin{verbatim}
Проанализируй данные рекламных кампаний:
Кампания А: бюджет 100K, конверсия 2.1%, CPA 450₽
Кампания Б: бюджет 80K, конверсия 1.8%, CPA 380₽
Кампания В: бюджет 120K, конверсия 3.2%, CPA 520₽

Какую масштабировать? Какую закрыть? Почему?
\end{verbatim}

\subsection*{Сценарий 3: Создание презентаций и докладов}
\textbf{Что умеет:} структура, текст слайдов, рекомендации по визуалу.

\textbf{Примеры задач:}
\begin{itemize}
\item Подготовить презентацию для совета директоров за 30 минут
\item Сделать доклад на конференцию — структура + текст
\item Создать тренинговые материалы для сотрудников
\end{itemize}

\textbf{Промпт-пример:}
\begin{verbatim}
Создай структуру презентации "Внедрение ИИ в HR".
10 слайдов. Аудитория: CEO и CFO.
Цель: получить бюджет 2M₽.
Каждый слайд: заголовок + 3 пункта + данные.
\end{verbatim}

\textbf{Как сделать красиво:} получите структуру → скопируйте в \textbf{Gamma.app} или \textbf{Tome} → AI нарисует слайды.

\subsection*{Сценарий 4: Генерация изображений}
\textbf{Что умеет:} создавать картинки по описанию, вариации, редактирование.

\textbf{Примеры задач:}
\begin{itemize}
\item Сделать иллюстрации для презентации
\item Создать баннеры для рекламы (10 вариантов за минуту)
\item Сгенерировать персонажа для обучающих материалов
\item Редактировать фото — заменить фон, добавить элементы
\end{itemize}

\textbf{Промпт-пример (для Kandinsky/Midjourney):}
\begin{verbatim}
Профессиональная иллюстрация: современный офис,
сотрудник работает с голограммой данных,
футуристичный стиль, синие и оранжевые тона,
высокая детализация, 16:9
\end{verbatim}

\subsection*{Сценарий 5: Глубокие исследования}
\textbf{Что умеет:} искать, анализировать, сравнивать, обобщать информацию из множества источников.

\textbf{Примеры задач:}
\begin{itemize}
\item Исследовать рынок конкурентов за 1 час вместо 1 недели
\item Сравнить 10 AI-инструментов — функции, цены, плюсы/минусы
\item Подготовить аналитическую записку по тренду отрасли
\item Найти научные статьи по теме и резюмировать
\end{itemize}

\textbf{Инструмент: Perplexity.ai} — ищет в интернете и выдаёт ответ с источниками. Заменяет Google + аналитика.

\subsection*{Сценарий 6: Написание кода и автоматизация}
\textbf{Что умеет:} писать скрипты, исправлять ошибки, объяснять код.

\textbf{Примеры задач:}
\begin{itemize}
\item Автоматизировать отчётность в Excel (VBA/Python)
\item Написать скрипт для обработки 1000 файлов
\item Создать Telegram-бота для уведомлений
\item Исправить ошибку в существующем коде
\end{itemize}

\textbf{Промпт-пример:}
\begin{verbatim}
Напиши Python-скрипт, который берёт все CSV-файлы
в папке, объединяет их в один Excel и добавляет
столбец "Источник" с названием файла.
\end{verbatim}

\subsection*{Сценарий 7: Распознавание и озвучка}
\textbf{Что умеет:} превращать речь в текст, текст в речь, музыку из текста.

\textbf{Примеры задач:}
\begin{itemize}
\item Расшифровать запись совещания (1 час → 5 минут чтения)
\item Озвучить обучающий курс (15 часов текста → реалистичный голос)
\item Создать подкаст из статьи
\item Сгенерировать фоновую музыку для видео
\end{itemize}

\textbf{Инструменты:} Whisper (OpenAI) — распознавание. ElevenLabs — озвучка. Suno — музыка.
