# SafeMind — Бесплатный гид по ИИ-инструментам
## Страница 1: Какие LLM существуют и что они умеют

### 1. ChatGPT (OpenAI)
Бесплатная версия: **ChatGPT 4o-mini** — безлимитно.
Платная: ChatGPT Plus $20/мес.

Что умеет бесплатная версия:
• Писать тексты любой сложности — от email до стратегий
• Создавать презентации (описание слайдов + структура)
• Анализировать CSV-данные — выдавать инсайты, графики (код Python)
• Генерировать изображения (DALL-E — ограниченно)
• Писать код и исправлять ошибки
• Переводить между 50+ языками
• Резюмировать длинные документы

### 2. Claude (Anthropic)
Бесплатная версия: **Claude.ai** — лимит запросов в день.
Платная: Claude Pro $20/мес.

Чем отличается от ChatGPT:
• Лучше работает с длинными документами (до 100 000 токенов)
• Более "вдумчивые" ответы на сложные вопросы
• Отлично пишет аналитические отчёты
• Может анализировать загруженные PDF и Excel

### 3. Gemini (Google)
Бесплатная версия: **Gemini 1.5 Flash** — безлимитно.
Платная: Gemini Advanced.

Особенности:
• Глубокая интеграция с Google Docs, Sheets, Gmail
• Может работать с видео и аудио
• Большой контекст — до 1 млн токенов
• Бесплатно анализирует PDF до 1500 страниц

### 4. YandexGPT (Яндекс)
Бесплатная версия: **YandexGPT Lite** через Яндекс.Браузер.
Платная: YandexGPT API.

Особенности:
• Лучше понимает русский язык и контекст
• Интеграция с Яндекс.Облаком
• Подходит для российского бизнеса
• Работает с таблицами и документами

### 5. Mistral / Llama (Open Source)
Бесплатно через HuggingFace, Perplexity, или локально.

Особенности:
• Полностью бесплатно, если запускать на своём компьютере
• Не требует VPN
• Llama 3 — качество близко к ChatGPT
• Mistral — лучшие open-source модели для Европы

## Страница 2: Как это использовать — пошагово

### Формулировка описаний (тексты)
Промпт для ChatGPT/Claude:
```
Напиши [тип текста] на тему [тема].
Аудитория: [кто читает].
Тон: [формальный / дружеский / экспертный].
Объём: [N слов / страниц].
Включи: [ключевые пункты].
```

Пример:
```
Напиши описание услуги для сайта.
Аудитория: HR-директора компаний 500+ человек.
Тон: экспертный, с цифрами.
Объём: 200 слов.
Услуга: внедрение AI в подбор персонала.
```

### Создание презентаций
Промпт:
```
Создай структуру презентации из 10 слайдов на тему [тема].
Для каждого слайда: заголовок, 3-5 ключевых пунктов, рекомендация по визуалу.
Целевая аудитория: [кто].
Цель презентации: [чего добиться].
```

Чтобы сделать красивую презентацию:
1. Получи структуру от ChatGPT
2. Скопируй в Gamma.app или Tome (бесплатно, AI-генерация слайдов)
3. Или используй Google Slides + плагин SlidesAI

### Анализ данных
CSV-файл → ChatGPT / Claude:
```
Проанализируй данные. Вот CSV:
[вставь содержимое файла]

Вопросы:
1. Какие тренды видишь?
2. Какие аномалии?
3. Построй прогноз на следующий месяц.
4. Предложи 3 действия на основе данных.
```

Или загрузи файл напрямую в Claude.ai — он читает Excel и PDF.

### Советы по промптам
✅ Конкретизируй: "Напиши email" → плохо. "Напиши email клиенту о задержке поставки, тон извиняющийся, с предложением скидки 10%" → хорошо.
✅ Давай контекст: кто читает, какая цель, какой формат.
✅ Проси формат: bullet points, таблица, email, код.
✅ Если ответ слабый — напиши "Сделай глубже" или "Добавь примеры из [индустрия]".
