## Страница 3: Решения для экономистов

### Как ИИ помогает экономисту

**1. Макроэкономический анализ**
• AI мониторит 100+ индикаторов в реальном времени
• Прогнозирование инфляции, ВВП, ключевой ставки
• Сценарный анализ: базовый, оптимистичный, пессимистичный
• Раннее предупреждение о кризисных сигналах

Промпт:
```
Проанализируй макроэкономическую ситуацию:
Инфляция: 7.5% (цель ЦБ: 4%)
Ключевая ставка: 16%
ВВП рост: +2.1%
Безработица: 2.8%

Вопросы:
1. Когда ожидать снижение ставки?
2. Какие риски для бизнеса в Q3?
3. Рекомендации по стратегии на год.
```

**2. Отраслевой анализ**
• AI анализирует отрасль — конкуренты, тренды, барьеры
• Сравнительный анализ компаний-лидеров
• Прогнозирование спроса по сегментам
• Мониторинг регуляторных изменений

**3. Инвестиционный анализ**
• Оценка проектов: NPV, IRR, срок окупаемости
• Анализ чувствительности — AI меняет переменные
• Портфельный анализ — диверсификация и риски
• Сравнение инвестиционных альтернатив

Промпт:
```
Оцени инвестиционный проект:
Инвестиции: $2M
Срок: 5 лет
Прогноз выручки: $800K → $1.5M → $2.2M → $3M → $3.5M
OPEX: 60% от выручки
Ставка дисконтирования: 12%

Рассчитай: NPV, IRR, срок окупаемости.
Анализ чувствительности: что если выручка -20%?
```

**4. Автоматическая отчётность**
• Еженедельные дайджесты экономических новостей
• Автоматические обновления прогнозов по данным
• Презентации для совета директоров — AI готовит слайды
• Перевод отчётов на английский / китайский
