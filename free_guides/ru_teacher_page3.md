# SafeMind — Бесплатный гид по ИИ-инструментам
## Страница 3: Решения для преподавателей

### Как ИИ помогает учителю

**1. Подготовка к занятиям**
• Генерация плана урока по любой теме за 2 минуты
• Создание раздаточных материалов и карточек
• Подготовка тестов с автоматической проверкой
• Адаптация материала под разные уровни (слабые/сильные)

Промпт:
```
Составь план урока географии, 6 класс.
Тема: "Жираф — житель саванн".
Формат: 45 минут.
Включи: вступление (5 мин), основная часть (30 мин),
практическое задание (7 мин), заключение (3 мин).
Добавь 5 вопросов для проверки.
```

**2. Проверка работ**
• AI проверяет тесты и задания с закрытыми ответами
• Анализ сочинений — структура, стиль, ошибки
• Генерация развёрнутых комментариев для учеников
• Выявление типичных ошибок класса — для групповой работы

**3. Индивидуализация**
• Персональные задания для каждого уровня
• AI-репетитор для слабых учеников (объясняет 10 раз, не устаёт)
• Генерация дополнительных материалов для сильных
• Планы коррекции под каждого ученика

Промпт:
```
Ученик 7 класса не понимает дроби.
Составь 3 задания начального уровня,
2 промежуточных, 1 продвинутое.
К каждому — подробное объяснение.
```

**4. Административная работа**
• Автоматические отчёты для администрации
• Генерация характеристик учеников
• Подготовка программ и методичек
• Перевод материалов на другие языки
