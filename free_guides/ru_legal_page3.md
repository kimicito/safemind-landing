# SafeMind — Бесплатный гид по ИИ-инструментам
## Страница 3: Решения для юристов

### Как ИИ помогает юристу

**1. Анализ документов**
• Проверка договоров на риски — за минуты, а не часы
• Сравнение версий документов — автоматический diff
• Поиск несоответствий в NDA, контрактах, соглашениях
• Извлечение ключевых условий из 100-страничного документа

Промпт:
```
Проанализируй договор на риски для заказчика.
Вот основные пункты:
[вставь текст договора]

Выдели:
1. Неблагоприятные условия
2. Отсутствующие стандартные защитные пункты
3. Неясные формулировки
4. Рекомендации по правкам
```

**2. Поиск прецедентов**
• AI ищет релевантные кейсы по описанию ситуации
• Сравнение текущего дела с аналогичными
• Прогнозирование вероятности успеха по статистике
• Автоматический мониторинг изменений в законодательстве

**3. Генерация документов**
• Стандартные договоры — по параметрам
• Исковые заявления — по фактам
• Правовые заключения — по запросу
• Перевод юридических текстов на другие языки

Промпт:
```
Состави исковое заявление о взыскании задолженности.
Сумма: 500 000 руб.
Основание: договор поставки №123 от 01.01.2024.
Срок оплаты: 30.03.2024 (просрочка 60 дней).
Доказательства: договор, акты, переписка.
```

**4. Консультирование**
• Первичный анализ обращений клиентов
• Подготовка ответов на типовые вопросы
• Анализ правовой позиции противной стороны
• Прогнозирование судебных издержек
